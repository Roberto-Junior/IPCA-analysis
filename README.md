# IPCA-analysis

## este projeto consiste em consumir dados do IPCA da API Sidra e realizar algumas analises. 
## Etapas do projeto:
## - Importar dados dos últimos 24 meses da API.
## - Acompanhar o comportamento da inflação acumulada em 12 meses.
## - Criar índice base 100 utilizando a variação mensal.
## - Calcular variação acumulada nos últimos 24 meses.

## Recomendo acessar este projeto por [este link](https://jovian.ai/roberto-junior/ipca) para visualizar os gráficos interativos que não são mostrados no GitHub.
